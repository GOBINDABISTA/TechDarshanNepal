# Tech दर्शन Website

Modern, responsive website for Tech दर्शन, a newly established company specializing in digital product development including websites, web apps, mobile apps, and digital solutions.

## Features

- Single-page design with smooth scrolling navigation
- Mobile-first responsive layout using CSS Flexbox and Grid
- Interactive elements using vanilla JavaScript (no external libraries)
- 3D animated neural network background on the hero section
- Dynamic dropdown navigation menus with multi-level support
- Interactive particles background effects on key sections
- Comprehensive sections:
  - Home/Hero with animated typing effect
  - About with company history and values
  - Services with expandable service cards
  - Client sectors with visual categorization
  - Success Stories with case studies and video content
  - Interactive testimonial slider
  - Comprehensive careers section with job listings and application form
  - Blog section with categorized posts and newsletter signup
  - Team showcase with social media links
  - Contact form with validation

## Technology Stack

- HTML5 for semantic structure
- CSS3 with Flexbox/Grid for responsive layouts
- Vanilla JavaScript for interactivity
- Font Awesome for icons
- Google Fonts for typography
- Custom animations and transitions

## Performance Optimizations

- Lazy loading of images for faster initial page load
- Optimized asset loading and compression
- Mobile performance considerations (reduced animations)
- SEO best practices implemented throughout

## Browser Compatibility

Tested and working on:
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+
- Mobile browsers (iOS Safari, Android Chrome)

## Setup and Installation

1. Clone the repository: `https://github.com/GOBINDABISTA/TechDarshanNepal.git`
2. Navigate to the project folder: `cd TechDarshanNepal`
3. Open index.html in your browser or use a local server
